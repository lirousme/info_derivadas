Um chat é feito por várias mensagens, que são perguntas e respostas.
Vamos supor que o personagem 1 mande uma mensagem perguntando:
Personagem 1: "O que é morango?"
O personagem 2 responde: "Morango é uma fruta."
Aqui temos um chat completo.
Mas vamos supor que o criador queira expandir esse conhecimento.
Um chat completamente novo será criado, nesse novo chat terá as duas mensagens do chat anterior ("O que é morango?", "Morango é uma fruta.") e continuará assim: "Qual a cor do morango?"
Mas podemos vazer outra derivação do mesmo chat 1: "O que é morango?", "Morango é uma fruta." e continuará assim: "Quais países comem mais morangos?"
Ou pode ter uma chat assim: "O que é morango?", "Morango é uma fruta.", "Qual a cor do morango?", "Vermelho" e continua assim: "Qual outra fruta também é vermelha?" "Maçã."

É como se um grupo de mensagens iniciais pudessem terminar de formas diferentes, então temos vários chats que começam com as mesmas mensagens, mas terminam com mensagens diferentes.
